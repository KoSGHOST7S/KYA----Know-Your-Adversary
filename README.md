# KYA -- Know Your Adversary

> K.Y.A. is a security hardening framework that takes offensive pentesting tradecraft and flips it into a blue team defensive stance. Every control maps to a real attack.

---

## What is KYA?

KYA is built on a simple premise --Readme the best person to lock a door is the one who knows how to pick it. Instead of generic hardening checklists, every defensive control in KYA traces back to a real-world exploitation technique drawn from offensive security methodologies including OWASP Top 10, CWES and HTB CPTS tradecraft.

If it's in the framework, someone has used it to break into a system just like yours.

---

## Features

- Offense-to-defense vector mapping based on real pentesting techniques
- Hardening controls organized by attack category (web, network, privilege escalation, etc.)
- Built from hands-on exploitation experience, not theory
- Designed for pentesters supporting blue team operations
